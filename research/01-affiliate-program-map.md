## NICHE 4 — AML / KYC / Compliance ❌ FAIL (do not build as affiliate)

**This was your day-job pick, and the data says no.** Compliance/RegTech is enterprise, usage-priced and sales-led: vendors want channel partners who *close deals* (MSPs, auditors, fractional CISOs), not content publishers driving self-serve signups. Verified count of public self-serve programs that clear the gate: **1** (SEON, and its rate is gated behind signup).

| Tool | Reality | 
|---|---|
| **SEON** | ✅ Only clear public affiliate (PartnerStack, 90-day cookie); rate gated behind signup |
| **Vanta** | $500 one-time refer-a-friend bounty — not a tracked content-affiliate program |
| **Drata** | No public self-serve program (application-only channel alliance) |
| **Secureframe / Sprinto / Thoropass** | Sales-led referral; "recurring" language but gated, MSP-oriented |
| **ComplyAdvantage / Sumsub / Jumio / Onfido / Trulioo / Persona / Alloy / Sanction Scanner** | 100% reseller/channel — no affiliate |
| **KnowBe4 / Hyperproof / AuditBoard** | Reseller / enterprise alliance only |

**SOC2/GRC** is the least-bad sub-niche, but it monetizes as a **referral-partner lead-gen play** (you apply, get approved, register leads, accept one-time/gated payouts) — *not* a passive affiliate site. Out of scope for a Phase-1 affiliate build.
